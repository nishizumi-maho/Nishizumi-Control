# Changelog

All notable changes to Dominant Control for iRacing.

## [3.0.0] - 2025-02-15
### Added
- Offline voice command pipeline using Vosk (downloadable models, configurable path, push-to-talk friendly).
- Comprehensive documentation refresh covering every option (setup, HUD, combos, timing, device management, TTS, Vosk).
- MIT licensing and contributor-friendly guidance.

### Changed
- Updated quick start and installation flow to highlight start-order best practices and source installation steps.
- Requirements clarified for Python-based runs (`requirements.txt`).
- Default documentation tone now focuses on accessibility, education, and transparent use.

### Fixed
- Clarified troubleshooting for FFB conflicts, telemetry connection steps, and HUD visibility.

## [1.0.0] - 2024-12-11
### Initial Release
- Standalone Windows build with CONFIG/RUNNING modes, per-control bindings, presets, combos, HUD overlay, timing profiles, and device management.
